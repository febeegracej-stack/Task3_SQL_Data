# SQL Sales Analytics using Chinook Database

## Project Overview

This project demonstrates the use of SQL for data analysis using the Chinook Database. Various SQL operations were performed to extract business insights from customer, invoice, employee, and music sales data.

## Objectives

* Query structured data using SQL
* Perform filtering and sorting operations
* Use aggregate functions for analysis
* Apply JOIN operations across multiple tables
* Create reusable Views
* Improve query performance using Indexes

## Tools Used

* SQLite
* DB Browser for SQLite
* GitHub

## SQL Concepts Demonstrated

### Data Retrieval

* SELECT
* WHERE
* ORDER BY

### Aggregation

* COUNT()
* SUM()
* AVG()
* GROUP BY

### Joins

* INNER JOIN
* LEFT JOIN

## Business Insights Generated

### Top Revenue Customers

Identified customers generating the highest revenue for the business.

### Genre Popularity Analysis

Analysed music genre sales to determine customer preferences.

### Customer Support Performance

Evaluated employee workload based on customer assignments.
